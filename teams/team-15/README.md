## Problem Statement

India loses nearly 16% of food grain post-harvest due to improper storage, handling, and lack of advisory. This system provides a GenAI-powered advisor that generates tailored post-harvest management plans for farmers.

## Solution Overview

A Streamlit-based web application where farmers input crop details, harvest quantity, storage conditions, and region. The system generates a personalized management plan covering:

* Storage recommendations
* Pest control measures
* Moisture management
* Transport and handling practices
* Government scheme eligibility

The output is available in both English and Kannada, with DOCX export functionality.

## RAG (Retrieval-Augmented Generation) Architecture

The system implements a complete RAG pipeline using ChromaDB for vector storage:

### Components:

* **Embedding Model:** Sentence Transformers (all-MiniLM-L6-v2)
* **Vector Store:** ChromaDB for persistent storage
* **Retriever:** Semantic search with similarity threshold
* **Context Integration:** Retrieved documents injected into LLM prompt

### RAG Workflow:

1. FAO dataset is chunked and embedded
2. Embeddings stored in ChromaDB with metadata (source, crop, pest)
3. On user query, relevant documents retrieved via semantic search
4. Retrieved context appended to LLM prompt for grounded generation
5. ChromaDB persistent storage enables fast subsequent lookups

### Benefits:

* Grounds LLM responses in actual FAO data
* Reduces hallucination in pest control recommendations
* Enables semantic search across 1000+ data points
* Persistent vector store eliminates re-embedding on each run

## Features

### 1. Structured Farmer Input Form

* Crop selection (Rice, Wheat, Maize, Pulses, Groundnut)
* Harvest quantity (kg)
* Region (District, State)
* Storage type
* Moisture percentage

### 2. Personalized Post-Harvest Management Plan

* 5-section detailed advisory generated by Groq LLM
* Location-specific recommendations for Karnataka/India context

### 3. Pest Risk Calendar

* Crop-specific pest data
* Peak season identification
* Risk levels (Severe, High, Medium, Low)
* Damage description and control methods

### 4. Government Scheme Eligibility Check

* AMIF Warehouse Scheme (25% subsidy)
* WDPSA Free Storage (3 months for Karnataka)
* PM-KISAN benefits
* Contact details for each scheme

### 5. Bilingual Output

* English version
* Kannada translation (IndicTrans2 / Google Translate fallback)

### 6. DOCX Export

* Complete report download
* Includes all sections and data tables

## Data Source

### FAO Food Loss and Waste Database (WDPSA)

The dataset (`data/Data.csv`) contains post-harvest loss percentages for various commodities from official FAO data.

**Sample Records:**

| Commodity    | Loss Percentage | Stage      | Year |
| ------------ | --------------- | ---------- | ---- |
| Wheat        | 1%              | Processing | 2022 |
| Maize (corn) | 0.5–2%          | Processing | 2022 |
| Rice         | 2.5%            | Processing | 2022 |
| Sorghum      | 0.5%            | Processing | 2022 |
| Barley       | 12%             | Processing | 2022 |

These values are used to:

* Calculate expected losses in kilograms
* Estimate financial impact (₹25/kg average price)
* Inform LLM prompt with data-driven context
* Display stage-wise loss breakdown in reports

## Installation

### Prerequisites

* Python 3.8 or higher
* Groq API Key

### Setup Steps

#### 1. Clone the Repository

```bash
git clone https://github.com/Sahana8866/hackathon_ruralAI.git
cd hackathon_ruralAI
```

#### 2. Create Virtual Environment

**Windows:**

```bash
python -m venv venv
venv\Scripts\activate
```

**Linux / Mac:**

```bash
python -m venv venv
source venv/bin/activate
```

#### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

#### 4. Configure API Key

Create file `.streamlit/secrets.toml`:

```toml
GROQ_API_KEY = "your-groq-api-key-here"
```

#### 5. Run Application

```bash
streamlit run app.py
```

## Usage

1. Open the application in your browser (default: http://localhost:8501)
2. Fill the farmer input form in the left sidebar:

   * Select crop type
   * Enter harvest quantity (kg)
   * Specify district and state
   * Choose current storage type
   * Set moisture percentage using slider
3. Click **"Generate Management Plan"**
4. Review results across five tabs:

   * **Management Plan** - 5-section detailed advisory
   * **Pest Calendar** - Crop-specific pest information with controls
   * **Govt Schemes** - Eligibility results with contact details
   * **Kannada** - Kannada translation of the management plan
   * **Download** - DOCX report export
5. Click **"DOWNLOAD FULL REPORT (DOCX)"** to save the report
