# Student Project Idea Generator and Feasibility Evaluator

## Team 14

### Problem Statement
Students often struggle to select projects aligned with their skills, resources, and timelines.

### Solution
An AI-powered system that generates:
- 5 ranked project ideas
- Feasibility evaluation
- Dataset recommendations
- Technology stack suggestions
- 4-week implementation roadmap

### Tech Stack
- Python
- Streamlit
- Groq API
- GitHub API
- Plotly

### Features
- Personalized project recommendations
- Feasibility scoring
- GitHub trend analysis
- Dataset suggestions
- Roadmap generation
